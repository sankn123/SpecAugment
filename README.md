### Requirements:
1. python3
2. librosa
3. libsndfile
4. audioread
5. ffmpeg
5. numpy
6. tensorflow
7. tensorflow_addons

### Usage:
```
main.py [--dir][--policy]
```

--dir    | path/to/dataset | default='./LibriSpeech/'\
--policy | augmentation policy to use from {'LB','LD', 'SS', 'SM'} | deafault='LD'

OR

```
refer to demo/demo.ipynb for jupyter notebook demo
```
